# Amanar harness

The canonical interface is
[`$amanar-scaffold`](skills/amanar-scaffold/SKILL.md), supporting explicit
`new`, `adopt`, and `audit` modes. Behavioral fixtures verify preservation and
proportionality without imposing deterministic prose.

The six workflow and scaffold instrument skills support Claude, Pi, and Codex;
`workflow/agent-eval` remains Codex-native.

The former fixed Copier generator is available only from signed tag
`legacy-fixed-v1.0.11`. See
[`docs/migrating-from-legacy-fixed.md`](docs/migrating-from-legacy-fixed.md).
